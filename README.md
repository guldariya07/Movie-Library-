# Movie Library — Личная кинотека

**Автор:** Гульдария Арсланова
№**Дата выполнения** 28.04.26

## Описание

Десктопное приложение для ведения личной коллекции фильмов.  
Позволяет добавлять, просматривать, фильтровать и удалять фильмы, а также сохранять коллекцию в JSON-файл.

## Функциональность

- ✅ Добавление фильмов (название, жанр, год, рейтинг)
- ✅ Отображение в таблице с прокруткой
- ✅ Фильтрация по жанру (частичное совпадение) и году (точное совпадение)
- ✅ Удаление выбранного фильма
- ✅ Сохранение и загрузка из JSON
- ✅ Валидация ввода:
  - Год: число от 1888 до 2030
  - Рейтинг: число от 0 до 10
  - Название и жанр не могут быть пустыми

## Запуск

python movie_library.py
